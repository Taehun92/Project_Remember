<h4>로그인/회원가입 페이지</h4>
<div>로그인 페이지는 심플한 UI를 제공하여 누구나 쉽게 접근 할 수 있게 하였으며, 금색의 포인트색상으로 고급스러움을 표현하였습니다.</div>
<div>회원가입은 모달창으로 step별로 분리, 당장에 작성이 어려운 경우 추후 마이페이지에서 수정이 가능하다는 부분을 언급하여 회원가입의 번거로움을 최소화 하였습니다.</div>
</div>

게시글 클릭 시 상세화면과 댓글을 작성할 수 있게 모달창이 나오며, 사이드 바에 탭들을 추가해 쉽게 페이지 이동할 수 있게 구현하였습니다.

<h2 style="border-bottom: 1px solid #d8dee4; color: #282d33;"> 🎇 프로젝트 후기 </h2>   
<div align="center">
React를 활용하여 SNS를 만들었습니다.  
</div>
<div>useState를 활용하여 렌더링 개념은 이해하였으나 같은 페이지 내에서 렌더링이 일어나야 하는 여러 조건들을 전부 충족 하게 만들어 내는데 조금 어려움이 있었습니다.</div>
<div>또한, 페이지 내부에서 직접 코드를 작성하는 부분을 최소화하였다면, 컴포넌트를 조금 더 세분화하여 만들었다면, 완성도 있는 SNS서비스가 되었을 것이라 생각합니다. </div>
<div>최초 설계시 원하는 방향과, 이에따른 DB를 만들어 놓고 작업을 진행 하였으나, 중간에 컬럼 추가 삭제 하는 경우도 발생하였으며 </div>
<div>코드의 재사용을 최대한 하고자 노력하였으나, 비슷한 호출을 하는 코드들이 많이 발생하게 되는 것 같아 아쉬웠습니다.</div>
</div>
